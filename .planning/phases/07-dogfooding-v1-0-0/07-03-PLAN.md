    Open a new Claude Code CLI session in this repo directory (so `.mcp.json` is picked up):

    1. Start Claude Code: `claude` (in the repo root)
    2. Confirm the MCP server loaded — you should see "version-check-mcp" listed in available tools, or run `/mcp` to list active MCP servers
    3. Call validate_version:
       - Ask: "Use the validate_version tool to check if npm package react version 18.3.1 exists"
       - Expected: `{exists: true, ...}` response
    4. Call get_latest_version:
       - Ask: "Use the get_latest_version tool to find the latest stable version of the npm package lodash"
       - Expected: a version string like `{version: "4.17.21", ...}` (not a hallucinated string)
    5. Test a miss with alternatives:
       - Ask: "Use validate_version to check if npm package react version 99.0.0 exists"
       - Expected: `{exists: false, alternatives: [...]}` with real alternative versions

    **Signs of failure:** "Tool not found", timeout, or responses that don't match actual npm versions.

    If the server fails to load, check:
    - `cat .mcp.json` — confirm path matches binary location
    - Run the binary manually: `echo '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2024-11-05","capabilities":{},"clientInfo":{"name":"test","version":"0"}}}' | ~/bin/version-check-mcp 2>/dev/null`
