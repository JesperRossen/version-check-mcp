Each adapter already has a cached internal method that fetches/parses the version list. Add a `Versions()` method to each that calls the internal method and extracts the string keys:

**NPM** (`internal/registry/npm/npm.go`):
```go
func (a *Adapter) Versions(ctx context.Context, pkg string, incPre bool) ([]string, error) {
    p, err := a.packumentFor(ctx, pkg, incPre)
    if err != nil { return nil, err }
    keys := make([]string, 0, len(p.Versions))
    for k := range p.Versions { keys = append(keys, k) }
    return keys, nil
}
```

**PyPI** (`internal/registry/pypi/pypi.go`):
```go
func (a *Adapter) Versions(ctx context.Context, pkg string, incPre bool) ([]string, error) {
    proj, err := a.projectFor(ctx, pkg, incPre)
    if err != nil { return nil, err }
    keys := make([]string, 0, len(proj.Releases))
    for k := range proj.Releases { keys = append(keys, k) }
    return keys, nil
}
```
